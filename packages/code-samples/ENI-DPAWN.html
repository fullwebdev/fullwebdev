Codes d'exemple

Vous retrouverez dans cette archives certains codes d'exemples issus de mon livre "Développement et architecture des Applications Web Modernes"édité en 2021 aux éditions ENI.

Les mini-projet de démonstration sont présentés par chapitre dans la liste ci-dessous. Cliquez sur le lien pour tester un projet dans votre navigateur.

La liste complète de fichiers sources utilisés dans le livre est donnée par ordre d'apparition. Cliquez sur la flèche à gauche du nom de chaque mini-projet pour faire apparaître le chemin vers tous les fichiers correspondants à partir du dossier src/. Le hash tag en fin de chemin indique la région du fichier (#region <clef>) présentée dans le livre.

Ressources en ligne

Cette archive ne contient qu'une part des codes d'exemples utilisés dans le livre, certains pouvant être trop complexes ou volumineux. Afin de rester précisément conforme au contenu imprimé, il ne recevra pas ou peu de mises-à-jour.

Pour consulté l'ensemble du code versionné et régulièrement mises-à-jour, ainsi que des contenus complémentaires, rendez-vous sur fullweb.dev.
